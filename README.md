# Bullet Game

BulletGame is a Racing Game developed by Marc Garcia and Arnau Gallofr√© in order to learn OpenGL and the Bullet library. The main goal of this game is to Push the ball towards the goal before the other player. Also you may have to avoid some obstacles and punish your opponent.

Github Link: https://github.com/ggaka6god/bullet

## Controls:

### Player 1 ( Red )
- W to accelerate.
- A and D to turn left or right respectively.
- S to brake.
- E to restart.

### Player 2 ( Green )
Same mechanics as the red player:
- Arrows movement instead of the W,A,S,D keys.
- Enter to restart.
